## âœï¸ Customization

### Change Your Name
Edit the logo text in `index.html`:
```html
<h1 class="logo-text">
  <span>Y</span><span>O</span><span>U</span><span>R</span><span>N</span><span>A</span><span>M</span><span>E</span>
```

### Change Colors
Update the CSS variables in `css/variables.css`:
```css
:root {
  --bg-primary: #F7F7F7;
  --text-primary: #1A1A1A;
  --text-secondary: #6F6F6F;
  --accent: #F97316;
}
```
